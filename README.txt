Pour lancer l'application il suffit de se placer dans le dossier et d'y avoir, au préalable créé le fichier de la base de connaissance.
Puis d'executer la commande :
ant -Dfile=<fichierQueVousSouhaitezTester>

Par exemple pour tester avec la base de connnaissance fournie :
ant -Dfile=baseDeConnaissance.txt
